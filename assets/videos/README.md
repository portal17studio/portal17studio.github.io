# Demo videos

Drop the real screen recordings here with these exact filenames and the
homepage will pick them up automatically — no HTML/JS changes needed:

- `demo-recording.mp4` — "Record your first flow"
- `demo-branching.mp4` — "Branching, loops & SQL"
- `demo-scheduler.mp4` — "Scheduling unattended runs"

Until a file exists, clicking that card's thumbnail opens a modal with a
friendly "coming soon" message instead of a broken video player.

Recommended format: MP4 (H.264 + AAC), 1080p or 720p, ideally under ~30-40MB
each so the page stays fast on GitHub Pages (which serves files as-is, with
no video transcoding). If a recording is bigger, consider trimming it or
hosting it elsewhere (e.g. YouTube) and swapping the card to link out instead
of opening the in-page modal.
